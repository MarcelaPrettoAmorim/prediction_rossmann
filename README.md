# Predição de vendas - Rossmann

A rede de farmácias **Rosmann** está iniciando um planejamento de custos para o ano seguinte. Para determinar quanto será destinado para custear cada loja (reformas, campanhas de marketing, novas contratações, aumento de estoque, entre outros custos), os tomadores de decisão precisam de uma **previsão** de quanto será a **receita de cada loja nas próximas 6 semanas** para estipular o valor que cada uma receberá, a fim de que os investimentos realizados tenham o maior retorno possível.

O projeto tem como objetivo realizar a previsão de faturamento das lojas da rede Rossmann utilizando técnicas de Machine Learning.

## 📄 Metodologia

Para a construção da solução será apliocada a metodologia CRISP-DM(Cross Industry Standard Process for Data Mining), que sugere as seguintes etapas:

- Entendimento do negócio
- Entendimento dos dados
- Preparação dos dados
- Modelagem
- Avaliação e validação
- Deploy do modelo (colocar em produção)
