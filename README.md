# Prediction Rossmann

Este projeto tem como objetivo prever as vendas futuras da rede de farmácias Rossmann com base em dados históricos.

## 📂 Estrutura do Projeto

- `data/` - Contém os datasets utilizados no projeto
- `notebooks/` - Jupyter Notebooks com as análises
- `src/` - Código-fonte do projeto
- `README.md` - Documentação do projeto

## 🚀 Como rodar o projeto

1. Clone o repositório:
   https://github.com/MarcelaPrettoAmorim/prediction_rossmann.git

2. Instale as dependências
   pip install -r requirements.txt

📊 Tecnologias usadas
Python
Pandas
Scikit-Learn
Matplotlib
TensorFlow
